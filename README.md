# 2025年河源市第二次数学建模比赛作品（程序源码）

### 代码运行要求:
1. 环境: python 3.11 及以上
2. 模块:
    - scikit-learn
    - numpy
    - pandas
    - matplotlib
3. uv
    - 程序运行请使用命令: `uv run main.py`
    - 如果不是cpython 3.11版本需要加上`-p <版本号>`, 即`uv run -p <版本号> main.py`
        - **Example**: `uv run -p 3.13 main.py`
